# Linuxify filenames
Make filenames more linux friendly.

### Install/uninstall:
**Install** the script with.
```
./install.sh
```

**Uninstall** the script with.
```
./uninstall.sh
```

### Usage:
```
linuxify <filename/s>
```

### Example formatting:
**ORIGNAL**: Hello    theRe.txt | **FORMATTED**: hello-there.txt
